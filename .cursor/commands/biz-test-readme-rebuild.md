# biz-test-readme-rebuild
- 目的：重构项目的readme的文档链接

- 任务执行时间：当前会话的所有任务都执行完成了，通常是所有文件都生成完成
- 步骤：
    1. 检查当前生成的文件是否有readme.md文件，如果有，在文件的最上面检查是否有文件生成时间，如果没有，添加上。时间格式要为：年月日 时分秒
    2. 再依次向上找是否有已经生成的readme.md文件，依次将子文件中的内容变动总结到readme.md中。（注意：越上层，应该越精简）
        - 在引用时，追加版本时间（被引用的readme的文件生成时间）在链接中。如果引入的链接时间和文件的readme.md的时间不一致，需要同步内容和时间
    3. 每层readme.md中都要在最上层补充最新更新时间
